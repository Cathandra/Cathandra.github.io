vue组件化

组件化相当于一个功能模块，是独立存在的，可复用的。它不单单有页面，还有业务逻辑和样式。

好处：可实现功能模块的复用，提高项目执行效率，适用开发复杂的页面应用，方便代码阅读和维护；
缺点：状态管理问题（vuex），多组件、多页面、复杂业务混合使用（vue-router）,组件间传参、事件管理（props、emit/on、bus）。
